# Jade Core Frontend 🧬📊

Jade Core es un ecosistema digital de alta precisión diseñado para converger dos áreas de especialización crítica: **Nutrición Clínica Renal** y **Consultoría en Ciencia de Datos e Inteligencia Artificial**. 

Este repositorio constituye la capa de presentación (Frontend) que orquesta la identidad de marca, la oferta de servicios y el acceso a recursos técnicos de alto nivel.

---

## 🚀 Propuesta de Valor
Jade Core materializa la **intervención clínica basada en evidencia** junto con la **automatización tecnológica avanzada**. Este frontend refleja una estética minimalista, profesional y "clínica", optimizada para una experiencia de usuario de alto rendimiento.

## 🛠️ Stack Tecnológico
El proyecto está configurado con **Vite** para un desarrollo ágil y eficiente:

* **Lenguajes:** `HTML5`, `CSS3`, `TypeScript`.
* **Herramientas de Construcción:** `Vite` (configurado en `vite.config.ts`).
* **Arquitectura:** Modular y organizada bajo tipado estricto (`tsconfig.json`).
* **Integración:** Gestión de dependencias profesional mediante `npm` (`package.json`).
